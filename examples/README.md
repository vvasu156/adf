# Azure Data Factory Examples

Use this folder to store sample ADF artifacts such as:

- pipeline JSON definitions
- linked service and dataset JSON
- ARM deployment templates
- example notebooks or scripts for ADF integration

## Suggested example files

- `copy-pipeline.json` – sample Copy Data pipeline
- `parameterized-pipeline.json` – pipeline with parameters
- `arm-template.json` – ARM template for deploying an ADF instance
