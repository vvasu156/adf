# Azure Data Factory Learning Repository

This repository is set up as a workspace for learning Azure Data Factory (ADF).

## What this repo includes

- `docs/learning-plan.md` – a guided learning plan for ADF concepts and hands-on practice
- `examples/` – a folder for sample pipelines, datasets, linked services, and deployment artifacts

## How to use this repo

1. Read `docs/learning-plan.md` to follow the suggested learning path.
2. Add your own ADF solution artifacts into the `examples/` folder.
3. Use Azure Data Factory in the Azure portal or VS Code to build and test pipelines.

## Recommended next steps

- Create an Azure Data Factory instance in your Azure subscription
- Connect ADF to source and sink data stores (Blob Storage, SQL, Data Lake)
- Build a sample pipeline and export the JSON to `examples/`
- Track notes, best practices, and sample artifacts in this repository
